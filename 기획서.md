
# Exit v1.0 – 최종 기획서 (완전판)

### 공통 커스텀 숫자 키보드
```
[음수/양수]  [+10만]  [+100만]  [+1000만]  [+1억]  [초기화]  (가로 스크롤)
[ 1 ] [ 2 ] [ 3 ]
[ 4 ] [ 5 ] [ 6 ]
[ 7 ] [ 8 ] [ 9 ]
[ . ] [ 0 ] [ ← ]
```
음수/양수 버튼은 화면 단위로 노출 제어 가능

### 온보딩 플로우 (4단계, 한 화면에 한 문항만 노출)
1 은퇴 후 희망 월 수입 디폴트 300만원 하단 안내: 2025년 1인 가구 최저생계비 143만원, 3인 가구 300만원
2 현재 순자산 (투자 가능 자산만) 디폴트 0원 상세 설명 포함 (증여 예정 포함, 실거주 부동산 제외 등)
3 월 평균 저축·투자 금액 (패시브인컴 재투자 포함) 디폴트 50만원
4 보유 자산 종류 (복수 선택, 나중에 수정 가능) 현금 / 예금·적금 / 주식 / 코인 / 채권 / 투자용 부동산 / 금 [완료하고 시작하기]

⠀→ 입력 후 확인 누르면 다음 문항 자동 포커스
### 고정 디폴트 값 (입력 안 받음)
* 은퇴 전 연 목표 수익률: 6.5%
* 은퇴 후 연 목표 수익률: 5.0%
* 예상 물가 상승률: 2.5%


### 홈화면 (첫 배포 시 유일한 화면)
```
[회사 탈출까지 7년 6개월 남았습니다.]
매월 200만원씩 넣으면
7년 6개월 후 일 안하고 월 300만원 놀고먹기 가능
[자세히 보기 ▼]

───────────────────────────────────
7,500만원 / 4억 2,750만원 (28%)

[시나리오 설정]  [기본 ✅] [물가폭등] [주식폭락] [로또당첨]  ← 가로 스크롤

[입금하고 기간 줄이기]          ← 큰 초록 버튼
[자산 변동 업데이트]            ← 작은 회색 버튼

───────────────────────────────────
안전 점수 총합 84점 ↑5점
목표 충족　　 18/25
수익률 안전성　25/25
자산 다각화　 21/25
자산 성장성　 20/25
```

### 목표 자산 계산 방식 (실질 4% 룰 + 월복리 시뮬레이션)
1. 은퇴 시 필요 자산 = (희망 월 수입 × 12) ÷ (은퇴 후 수익률 – 물가 상승률)
2. 현재 자산에서 월 투자금 + 월복리(은퇴 전 수익률) 적용 → 필요 자산 도달까지 개월 수 계산
3. 개월 수 → “X년 Y개월” + 진행률 % 변환  
→ 온보딩 종료 즉시 홈화면에 정확한 D-DAY 표시

### 안전 점수 4개 항목 (각 25점, 총 100점)
1. 목표 충족  (이번 달 패시브인컴 × 12) ÷ 희망 연 수입
2. 수익률 안전성 연환산 패시브인컴 ÷ 현재 총자산 → 3~8% 구간 25점
3. 자산 다각화  보유 자산군 개수 (7종 중 선택)
4. 자산 성장성 (이번 달 총자산 – 이전 달 총자산) ÷ 이전 총자산 – 2.5%

### 안전 점수 데이터 수집 방법 (기존 2개 버튼만 사용)
[입금하고 기간 줄이기] 버튼
* 기존: 이번 달 투자·저축 입금액 입력
* 추가 1줄 (선택 입력): “이번 달 받은 패시브인컴 총액” (배당+이자+월세 등) → 목표 충족, 수익률 안전성 계산에 사용

⠀[자산 변동 업데이트] 버튼
* 기존: 현재 총 투자 가능 자산 입력 (이전 값 자동 채움) → 자산 성장성 계산
* 하단 숨김 체크박스: 주요 자산군 7개 복수 선택 (최초 1회 후 거의 변경 없음) → 자산 다각화 계산

⠀→ 월 1회, 두 버튼만 눌러도 D-DAY 단축 + 안전 점수 4개 모두 실시간 갱신


### 시나리오 설정 (홈화면 상단에서 바로 접근)
홈화면의 [시나리오 설정] 또는 가로 스크롤 탭 클릭 → 풀스크린 시트 오픈

```
시나리오 선택 및 편집

[ 기본 ✅ ] [ 물가폭등 ] [ 주식폭락 ] [ 로또당첨 ] [ + 새 시나리오 ]

현재 선택: 기본

은퇴 후 희망 월 수입    300만원
현재 순자산        7,500만원
매월 목표 투자금액    200만원
은퇴 전 연 목표 수익률  6.5%
은퇴 후 연 목표 수익률  5.0%
예상 물가 상승률    2.5%

[복제하기] [이름 바꾸기] [삭제하기]
[저장하고 적용하기]
```

- 시나리오 변경 즉시 홈화면의 D-DAY, 목표 금액, 진행률 %, 안전 점수 전부 재계산되어 반영
- 최대 10개까지 저장 가능
- 가로 스크롤 탭은 저장된 시나리오 이름으로 실시간 동기화
